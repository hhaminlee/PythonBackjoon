# [백준] 9076 점수 집계

## 풀이 아이디어
- **핵심 방법**: 정렬 + 배열/리스트
- **접근 방식**: 배열/리스트를 활용한 구현 접근

## 복잡도 분석
**시간복잡도: O(n log n)**
**공간복잡도: O(1) (정렬 인플레이스 기준)**
- 정렬: O(n log n)

## 파이썬 코드
```python
import sys
input = sys.stdin.readline

n = int(input())

for _ in range(n):
    score_list = list(map(int, input().split()))
    score_list.sort()
    if score_list[3] - score_list[1] >= 4:
        print("KIN")
    else:
        print(sum(score_list[1:4]))
```
